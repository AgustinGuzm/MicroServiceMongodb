MONGO DATABASE CONNECTION DEMO
Mongo Database. MongoDB is a source-available cross-platform document-oriented database program. Classified as a NoSQL database program, MongoDB uses JSON-like documents with optional schemas. The pourpuse of little app is to demostrate that we can connect a non-relational database usgin the Java Spring Boot with RestController to connect it. Also, we use Agular as a front end As a good practice, in this app you can see how to apply angular's templates that help to improve the front presentation, Also you can check just a little implementation of CSS
